# ✈️ PilotLog - Inteligentny Dziennik Lotów

**PilotLog** to aplikacja mobilna na platformę Android, stworzona jako projekt studencki. Aplikacja zastępuje tradycyjny papierowy logbook pilota, oferując nowoczesne funkcjonalności takie jak automatyczne śledzenie czasu lotu, integrację z danymi pogodowymi i monitorowanie przeciążeń.
